This project is a fully functional Art Gallery Visitor Management System built using Java, applying core Object-Oriented Programming (OOP) concepts such as Abstraction, <br> Inheritance, Encapsulation, and Polymorphism, along with a graphical user interface built using Java Swing. <br>
The system allows an art gallery to register visitors, manage ticket types, track visits, handle purchases and cancellations, assign personal art advisors, calculate discounts and rewards, and store visitor activity through file handling.

## Technology Used
<ul>
  <li>Java's OOP concept</li>
  <li>Java Swing and AWT</li>
  <li>File handling using buffered reader and buffered writer</li>
  <li>Exception handling</li>
</ul>
